# Extension Icons

Place your extension icons here:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

For development, you can create simple placeholder icons or use an online icon generator.

## Creating Simple Icons

You can use any image editor or online tool like:
- https://favicon.io/favicon-generator/
- https://www.canva.com/

The icon should represent focus/productivity (e.g., a shield, timer, or focus symbol).
